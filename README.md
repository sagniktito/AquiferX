## Aquifer Modeling & Reservoir Drive Analysis Toolkit


AquiferX is an open-source Python-based reservoir engineering toolkit designed for:

- Water influx modelling
- Material balance analysis
- Reservoir drive identification
- Reservoir performance diagnostics


## Features


### Current

- Streamlit reservoir dashboard
- Pressure performance visualization


### Planned


### Material Balance

- Material Balance Equation
- Havlena-Odeh diagnostics
- OOIP estimation
- Drive indices


### Aquifer Models

- Schilthuis steady-state model
- Pot aquifer model
- Carter-Tracy model

Future:

- Fetkovich aquifer model
- Van Everdingen-Hurst model


## Technology

Python

Streamlit

NumPy

SciPy

Pandas
